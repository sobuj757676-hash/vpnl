# Website Redesign Prompts

এই ফোল্ডারে ( `docs/redesign_prompts/` ) ওয়েবসাইটটির মেন পেজ রিডিজাইন করার জন্য ৪টি আলাদা প্রম্পট দেওয়া আছে। যেহেতু AI-কে একবারে বড় টাস্ক দিলে সাধারণত কোড সম্পূর্ণ বা নির্ভুলভাবে সম্পন্ন করতে পারে না, তাই প্রম্পটগুলো ধাপে ধাপে (step-by-step) সাজানো হয়েছে।

## ব্যবহারের নিয়ম (How to use with an AI Coding Assistant)

1. **প্রম্পট ১ দিন:** প্রথমে `01_header_and_hero.md` ফাইলের সম্পূর্ণ লেখাটি কপি করে AI-কে দিন।
2. **অপেক্ষা করুন:** AI কোড আপডেট করবে। কাজ শেষ হলে AI অপেক্ষা করবে।
3. **নেক্সট কমান্ড:** AI-কে শুধু `next` বা `পরের ধাপে যান` বলুন, এবং সাথে `02_trust_and_features.md` এর প্রম্পটটি দিন।
4. **এভাবে চালিয়ে যান:** একইভাবে ৩য় এবং ৪র্থ প্রম্পটগুলো ( `03_use_cases_catalog.md` এবং `04_stats_about_footer.md` ) দিয়ে পুরো ওয়েবসাইটের রিডিজাইন সম্পন্ন করুন।

এই প্রম্পটগুলোতে আধুনিক Framer Motion এনিমেশন, গ্লাস ইফেক্ট (Glassmorphism), এবং বেন্টো গ্রিড (Bento Grid) এর মতো টপ-টায়ার (Top-tier) ওয়েবসাইট ডিজাইন প্যাটার্নের নির্দেশ দেওয়া আছে, যা AI সহজেই বুঝতে পারবে।
