# Website Redesign Prompts (Top-Notch Tier)

এই ফোল্ডারে ( `docs/redesign_prompts/` ) ওয়েবসাইটটির মেন পেজটিকে একেবারে **Top-Tier / Next-Level SaaS** (যেমন Vercel, Linear, Stripe) এর মতো রিডিজাইন করার জন্য ৪টি আলাদা প্রম্পট দেওয়া আছে।

যেহেতু এই ডিজাইনগুলোতে অত্যন্ত অ্যাডভান্সড কোডিং (যেমন: Framer Motion এর `useScroll`, `useTransform`, 3D Card Tilt, Linear-style mouse spotlight) ব্যবহার করা হবে, তাই AI-কে একবারে পুরো পেজের টাস্ক দিলে সে ঠিকমতো কমপ্লিট করতে পারবে না। এজন্য প্রম্পটগুলো ধাপে ধাপে (step-by-step) অত্যন্ত নিখুঁতভাবে সাজানো হয়েছে।

## ব্যবহারের নিয়ম (How to use with an AI Coding Assistant)

1. **প্রম্পট ১ দিন:** প্রথমে `01_header_and_hero.md` ফাইলের সম্পূর্ণ লেখাটি কপি করে আপনার AI (যেমন Cursor, ChatGPT, বা Claude) কে দিন।
2. **অপেক্ষা করুন:** AI কোড আপডেট করবে এবং কাজ শেষ হলে পরবর্তী নির্দেশের জন্য অপেক্ষা করবে।
3. **নেক্সট কমান্ড:** AI-কে শুধু `next` বা `পরের ধাপে যান` বলুন, এবং সাথে `02_trust_and_features.md` এর প্রম্পটটি দিন।
4. **এভাবে চালিয়ে যান:** একইভাবে ৩য় এবং ৪র্থ প্রম্পটগুলো ( `03_use_cases_catalog.md` এবং `04_stats_about_footer.md` ) দিয়ে পুরো ওয়েবসাইটের রিডিজাইন সম্পন্ন করুন।

এই প্রম্পটগুলোতে যা যা আছে:
* **Framer Motion Mastery:** স্ক্রল-লিংকড অ্যানিমেশন, থ্রিডি কার্ড টিল্ট (3D Tilt), ম্যাগনেটিক বাটন।
* **Advanced UI Patterns:** বেন্টো গ্রিড (Bento Grid) মাউস স্পটলাইট ইফেক্টসহ, ইনফিনিট লোগো মার্কুই (CSS Masking), অ্যাপল-স্টাইল স্টিকি স্ক্রল।
* **Immersive Aesthetics:** গ্লাস ইফেক্ট (Glassmorphism), মেস/অরোরা গ্রেডিয়েন্ট, এবং প্যারাল্যাক্স ডেপথ।
